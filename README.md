# NotesAPP: Simple and Efficient Note Taking

**NotesAPP** is a simple and efficient application designed to help users create, manage, and organize their notes. It provides a clean and user-friendly interface for creating, editing, and deleting notes.

## Features

* **Create Notes:** Add new notes with titles and content.
* **Edit Notes:** Modify existing notes.
* **Delete Notes:** Remove notes when they are no longer needed.
## Technologies Used

* **Flutter:** For building the cross-platform mobile application.
* **Dart:** Programming language for Flutter.
* **Hive:** Local storage package for persisting notes on the device.

## Installation

**Prerequisites:**

Before running the app, ensure you have the following installed:

* Flutter SDK: https://docs.flutter.dev/get-started/install
* Dart SDK: https://dart.dev/get-dart

**Steps:**

### Step 1: Clone the Repository

   ```bash
   git clone [https://github.com/Youmna-Refaat/NotesAPP.git](https://github.com/Youmna-Refaat/NotesAPP.git)
```
### Step 2: Navigate to the project directory

   ```bash
cd NotesAPP
```
### Step 3: Install the dependencies
   ```bash
flutter pub get
```
### Step 4: Configure Firebase

*Set up Firebase for your Flutter app according to the guide: https://firebase.google.com/docs/flutter/setup
*Ensure you've configured Firebase Authentication and Firestore.

### Step 5: Run the app
  ```bash
flutter run
```
## Usage

Once the app is launched, you can:
     * **Create Notes:** Add new notes by entering a title and content.
* **Manage Notes:** View, edit, or delete existing notes.
